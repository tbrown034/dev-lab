Newsroom Data Visualization — Quick Reference

D3.js — Key Points
  • Not a charting library — a data binding toolkit. You build charts from building blocks.
  • Core concept: bind data to DOM elements. Each shape's size/position/color is driven by data.
  • Scales convert data values to visual values (pixels, colors). Domain = data, Range = visual.
  • Enter-update-exit handles live data: new items enter, existing update, removed exit. Perfect for election night.
  • geoPath + projection = Texas county maps. GeoJSON defines boundaries, D3 draws them as SVG paths.
  • SVG is just HTML — rect, circle, path, text. Inspect any D3 chart and it's regular markup.
Django — Key Points
  • Request lifecycle: URL → View → ORM query → Template (or JSON) → Response
  • Models = Python classes that map to database tables. Never write SQL for schema.
  • ORM queries: Story.objects.filter(published=True).order_by('-pub_date')
  • Admin panel = auto-generated web UI for editors. Register a model, get full CRUD. Newsroom killer feature.
  • DRF (Django REST Framework) = JSON APIs. Serializer + ViewSet + Router = full API.
  • Management commands = data pipelines. python manage.py import_election_results
  • Migrations = version-controlled database schema changes. Everyone stays in sync.
Election Night Pipeline

AP Feed → Django mgmt command (cron, every 2-5 min) → PostgreSQL (update_or_create) → DRF API (JSON endpoints) → D3 frontend (county map, enter-update-exit) → Reader (polls API, live updates)

Django admin = emergency fallback if pipeline breaks at 11pm.

AI + Elections — Your Edge
  • Anomaly detection: flag unusual vote swings or data errors in real-time
  • Auto alt text: generate meaningful chart descriptions for accessibility
  • Natural language summaries: "Republicans flipped 3 border counties for the first time since 1972"
  • Data cleaning: standardize names/formats across hundreds of counties or districts
  • Story discovery: surface unusual patterns in campaign finance data
  • Conversational explorer: readers ask "How did my county vote?" in plain English
The Journalist-Developer Edge

What transfers from journalism: Editorial judgment, FOIA/data skills, deadline instincts, knowing what makes a story, source verification, audience awareness.

Technical toolkit: Python, PostgreSQL, React/Next.js, API design, AI tools.

Why it matters: The code skills can be learned quickly. Editorial judgment takes a career to build.
