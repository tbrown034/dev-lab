## Key D3 Modules (v7 - modular architecture)
| Module | Purpose | Docs |
|--------|---------|------|
| d3-selection | DOM manipulation & data binding | https://github.com/d3/d3-selection |
| d3-scale | Mapping data to visual channels | https://github.com/d3/d3-scale |
| d3-axis | Axis rendering | https://github.com/d3/d3-axis |
| d3-shape | Lines, areas, arcs, pies | https://github.com/d3/d3-shape |
| d3-geo | Projections, paths, maps | https://github.com/d3/d3-geo |
| d3-transition | Animations | https://github.com/d3/d3-transition |
| d3-fetch | Data loading (csv, json, etc.) | https://github.com/d3/d3-fetch |
| d3-format | Number formatting | https://github.com/d3/d3-format |
| d3-time-format | Date formatting | https://github.com/d3/d3-time-format |
| d3-array | Array utilities (max, min, extent, bin) | https://github.com/d3/d3-array |
| d3-color | Color manipulation | https://github.com/d3/d3-color |
| d3-interpolate | Value interpolation | https://github.com/d3/d3-interpolate |
| d3-zoom | Zoom & pan behavior | https://github.com/d3/d3-zoom |
| d3-brush | Area selection | https://github.com/d3/d3-brush |
| d3-force | Force-directed layouts | https://github.com/d3/d3-force |
| d3-hierarchy | Tree, treemap, pack layouts | https://github.com/d3/d3-hierarchy |
