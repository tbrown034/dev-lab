Exercise 01: Hand-Code SVG Charts

No JavaScript! Write SVG directly in this HTML file. Open it in your editor and add shapes.

← Back to lesson

1. Vertical Bar Chart

Create 4 vertical bars representing these states. Bars should grow UPWARD from a baseline at y=250. Remember: taller bars = smaller y values.

Data: California (39.5M), Texas (29.1M), Florida (21.5M), New York (20.2M)

Hint: A bar with height 200 starting at y=50 reaches from 50 to 250 (the baseline).

2. Add a Reference Line

Add a horizontal dashed line showing a "national average" at roughly 15M (map that to a y-value yourself). Label it.

3. Color Each Bar Differently

Use these colors: #f97316, #818cf8, #4ade80, #f472b6
