15 Production Use Cases

Every card below contains a live, working D3 mini-visualization. These are the same techniques used by the New York Times, FiveThirtyEight, Bloomberg, and other newsrooms.

01 / GEO

Election Results Map

NYT, AP, CNN
  • TopoJSON parsed into GeoJSON feature collection
  • d3.geoAlbersUsa projection for US state outlines
  • Categorical color scale: red/blue by party
d3.geoPath topojson d3.geoAlbersUsa
02 / BAR

Live Results Bar Chart

Broadcast TV, AP, Decision Desk
  • Horizontal bars with animated width transitions
  • Vote counts update on a timer to simulate live data
  • d3.transition for smooth value interpolation
d3.scaleBand d3.transition enter/update
03 / LINE

Polling Average Line Chart

FiveThirtyEight, RCP, The Economist
  • d3.line generator with d3.curveBasis smoothing
  • Confidence band via d3.area with upper/lower bounds
  • Time scale on x-axis for date-based data
d3.line d3.area d3.curveBasis
04 / ARC

Election Needle / Gauge

New York Times
  • d3.arc generator for the gauge background track
  • Animated needle rotation via d3.interpolate on angle
  • Probability text updates in sync with needle position
d3.arc d3.interpolate transitions
05 / GEO

Choropleth Heatmap

WaPo COVID Tracker, CDC, Johns Hopkins
  • Sequential color scale (d3.interpolateYlOrRd) for intensity
  • Same geo pipeline as election map, different encoding
  • Quantize scale bins continuous data into discrete colors
d3.scaleSequential d3.geoPath topojson
06 / LAYOUT

Treemap

ProPublica, Bloomberg, Observable
  • d3.treemap computes nested rectangles from hierarchy
  • d3.hierarchy + sum() to roll up values
  • Color encodes category, size encodes dollar amount
d3.treemap d3.hierarchy d3.treemapSquarify
07 / ANIMATION

Animated Bar Chart Race

Bloomberg, Financial Times, Flourish
  • Bars re-sort and re-position with keyed data joins
  • d3.transition for smooth rank reordering
  • Timer-driven frames to animate over time
d3.transition data join (key) d3.interval
08 / ARC

Donut / Pie Chart

Campaign Finance, OpenSecrets, FEC
  • d3.pie computes start/end angles from data
  • d3.arc renders each slice as an SVG path
  • Animated entrance with attrTween on arc angles
d3.pie d3.arc attrTween
09 / SIMULATION

Force-Directed Graph

Network Analysis, Social Graphs, Knowledge Maps
  • d3.forceSimulation with charge, center, and link forces
  • Nodes repel each other, links pull connected pairs
  • Tick function updates positions every animation frame
d3.forceSimulation d3.forceLink d3.forceManyBody
10 / AREA

Stacked Area Chart

Budget Trackers, CBO, USASPENDING
  • d3.stack transforms tabular data into layered series
  • d3.area generator draws each layer as a filled shape
  • Color scale differentiates budget categories
d3.stack d3.area d3.scaleOrdinal
11 / SCATTER

Scatter Plot

Census Bureau, Pew Research, Gapminder
  • Two linear scales map x/y data to pixel coordinates
  • Circle radius encodes a third variable (population)
  • Color encodes category (region) via ordinal scale
d3.scaleLinear d3.scaleSqrt circle elements
12 / GRID

GitHub-style Heatmap

GitHub, GitLab, Contribution Calendars
  • Grid of small rects, one per day, colored by value
  • d3.scaleQuantize maps continuous values to color steps
  • d3.timeWeek and d3.timeDay compute grid positions
d3.scaleQuantize d3.timeWeek rect grid
13 / PATTERN

Small Multiples

NYT, FiveThirtyEight, The Economist
  • Same chart template repeated for each data subset
  • Shared scales ensure comparability across panels
  • SVG groups positioned in a grid layout
d3.line nested data shared scales
14 / GEO

Bubble Map

NYT, Reuters, CDC City-Level Data
  • Circles placed at projected [lon, lat] coordinates
  • d3.scaleSqrt sizes bubbles proportional to population
  • Layered on top of a base geographic path layer
d3.geoAlbersUsa d3.scaleSqrt projection
15 / INLINE

Sparklines

Dashboards, Stock Tickers, KPI Panels
  • Tiny line charts without axes, just the trend shape
  • d3.line with scaleLinear fit to a small viewport
  • Endpoint dot highlights the current/latest value
d3.line d3.scaleLinear minimal chrome
