Project 01

Global Temperature Anomalies (1880 – 2024)

NASA GISS surface temperature anomaly data visualized as a gradient area chart, showing the unmistakable warming trend with blue-to-red color encoding.

d3.area() d3.scaleSequential() linearGradient bisector tooltip

How I Built This

  • Used d3.area() with curveMonotoneX for smooth, natural-looking curves that don't overshoot the data
  • Applied an SVG <linearGradient> mapped to the x-axis, transitioning from blue (#3b82f6) to red (#ef4444) to encode the warming trend visually
  • Implemented hover tracking with d3.bisector() to find the nearest data point and display a tooltip with year and anomaly value
  • Added a zero-line reference and decade labels using d3.axisBottom with .tickValues() for clean, editorial-style labeling
  • The chart uses viewBox for full responsiveness, scaling from mobile to desktop without any JavaScript resize handlers
  • Entrance animation uses d3.transition() with a clip-path reveal, drawing the chart left-to-right like a broadcast wipe
Project 02

Federal Budget Breakdown (FY 2024)

Interactive treemap of U.S. federal spending. Click any category to zoom in and explore subcategories. Real data from the Congressional Budget Office.

d3.treemap() d3.hierarchy() zoom transitions nested data d3.format()

How I Built This

  • Structured budget data as a nested hierarchy and used d3.hierarchy() with .sum() and .sort() to compute the layout
  • Applied d3.treemap().tile(d3.treemapSquarify) for the squarified layout that maximizes readability of labels
  • Click-to-zoom uses d3.transition() to smoothly interpolate positions and sizes, re-running the treemap layout on the clicked node's children
  • Color scheme uses d3.scaleOrdinal() mapped to top-level department categories, with opacity variation for subcategories
  • Dollar formatting via d3.format("$.2s") converts raw billions to human-readable labels like "$1.2T" and "$340B"
  • Breadcrumb navigation lets users click back to the root view, a UX pattern borrowed from file explorers
Project 03

Media Market Viewership Dashboard

Top 20 DMA markets by TV households with a companion daypart viewership donut chart. Built to showcase broadcast industry data literacy.

d3.scaleBand() d3.arc() d3.pie() multi-chart layout staggered animations

How I Built This

  • Horizontal bar chart uses d3.scaleBand() for DMA market names and d3.scaleLinear() for household counts, with staggered .delay((d, i) => i * 40) entrance
  • Donut chart built with d3.pie() and d3.arc(), using innerRadius at 60% of outerRadius for the classic broadcast donut style
  • Both charts share a coordinated color palette and are laid out side-by-side using SVG transform groups
  • Nielsen DMA data is hardcoded from publicly available market rankings, representing real TV household penetration numbers
  • Polyline labels on the donut chart use d3.arc().centroid() for anchor points, preventing text overlap on small slices
  • Hover interactions highlight the corresponding bar or arc with opacity transitions, providing visual feedback
Project 04

Election Night Results Tracker

Styled after a broadcast TV election night graphic. Animated vote count bars with "Projected Winner" callouts when a candidate crosses the threshold. Uses real 2024 presidential election data for 10 swing states.

animated transitions conditional styling broadcast graphics d3.format() threshold logic

How I Built This

  • Each state is a paired bar group showing both candidates, using d3.scaleBand() with inner padding for the election-night look
  • Vote bars animate from zero using d3.transition().duration(1500) with d3.easeExpOut, simulating the vote-counting feel of live broadcasts
  • Staggered state-by-state reveals use .delay((d, i) => i * 200), creating a dramatic cascade like actual election night coverage
  • "Projected Winner" banners appear via opacity transition when a candidate's bar crosses 50%, checking margins for competitive vs. blowout states
  • Color scheme uses broadcast-standard red (#dc2626) and blue (#2563eb) with luminance-adjusted text for readability
  • Vote counts formatted with d3.format(",") for comma-separated thousands, matching network chyron style
