Learn Pandas

Pandas is how Python works with tabular data — the spreadsheet, the CSV, the SQL result set — all loaded into memory and reshaped with code. If you know SQL or JavaScript array methods, most of it will feel familiar fast.

Lessons are being written. In the meantime, the AI tutor on this page is live — ask it anything about DataFrames, cleaning data, grouping, or merging, and it will explain through SQL and JS analogies.
