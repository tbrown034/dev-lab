Sample Python

By the end of the first lessons, code like this will read like English.

# A tiny script: summarize a list of numbers
def summarize(numbers):
    # Like: numbers.filter(n => n > 0)
    positives = [n for n in numbers if n > 0]
    return {
        "count": len(positives),
        "total": sum(positives),
    }

print(summarize([3, -1, 7, 0, 5]))
# {'count': 3, 'total': 15}
